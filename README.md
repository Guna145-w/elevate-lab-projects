# 🤖 AI Projects Collection

A collection of AI-powered web applications built using **Google Gemini AI** to solve real-world problems in education and career development. This repository showcases two AI-integrated web applications developed using modern web technologies.

---

# 📚 AI Study Notes Generator

The AI Study Notes Generator helps students create well-structured study notes from any topic. Users simply enter a topic and optional instructions, and the application generates clear, organized, and exam-focused notes using Google Gemini AI.

### Features
- 📝 Generate AI-powered study notes
- 📖 Topic-based learning
- 🎯 Exam-oriented summaries
- 📋 Copy generated notes
- 📥 Download notes
- 🌙 Dark Mode
- 📱 Responsive Design

### Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
- Express.js
- Google Gemini API
- Dotenv

# 📄 AI Resume Analyzer

The AI Resume Analyzer evaluates resumes and provides intelligent feedback to help users improve their resumes for recruiters and Applicant Tracking Systems (ATS). It analyzes skills, experience, formatting, and overall resume quality using Google Gemini AI.

### Features
- 📄 Resume Analysis
- 🤖 AI-powered Feedback
- ✅ ATS Compatibility Suggestions
- 💡 Skill Assessment
- 📈 Resume Improvement Tips
- 🌙 Dark Mode
- 📱 Responsive Design

### Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
- Express.js
- Google Gemini API
- Dotenv
